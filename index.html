Some of my personal projects

  • SzimatSzatyor - SzimatSzatyor is a WoW (World of Warcraft) injector sniffer written in C++.
  • SmarterZip - Simple web client for Zipato smart homes. Designed to improve usability over the official interfaces.
  • Simple TPB Client - Alternative The Pirate Bay (TPB) client / website for searching and downloading torrents.
  • gsheet-shared-expenses-inserter - A simple web app for managing shared expenses using a Google Sheet as the backend.
  • WhatNext nCore - Simple browser addon for whatnext.cc The addon changes a domain in a link an also introduces a new feature.
  • RCT - RCT (Random Capital Tickers) contains Finviz screeners links.
  • RCIU - RCIU (Random Capital Instrument Updates) contains a table that contains instrument updates (added or removed).
  • RCNIS - RCNIS (Random Capital Netboon Instrument Searcher) is a browser addon that allows you to search for an instrument by its ticker.
  • Koronasok - Koronasok is a simple website for tracking infections of Coronavirus disease (COVID-19) in Hungary.
  • winston-json-log-formatter - A Node.js module that creates a console transport (by default) for winston to log messages in JSON format.
  • Redis CLI CS - Redis CLI CS is a C++ Redis CLI client connection string (URI) connector.
  • ini2json - ini2json is a simple INI to JSON converter written in C++.
  • MD5 Hasher - MD5 string/file hasher written in C++.
